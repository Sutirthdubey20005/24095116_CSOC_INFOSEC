# 🛡️ OverTheWire: Bandit Walkthrough (Levels 0 to 6)

This repository contains my write-up for **OverTheWire Bandit** levels 0 to 6, created as part of the **CSOC COPS 2025 - InfoSec Vertical**.

It walks through each level's objective, commands used, and the resulting password. This helps beginners build confidence with Linux commands, file operations, and basic hacking techniques.

---

## 🎮 What is Bandit?

[Bandit](https://overthewire.org/wargames/bandit/) is a wargame designed to teach the basics of Linux and security. It's the recommended starting point for anyone new to command-line and cybersecurity.

To connect to the game:
